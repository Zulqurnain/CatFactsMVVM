# CatFactsMVVM

An Android app demonstrating MVVM architecture with clean code principles, fetching cat facts from a REST API.

## Architecture & Tech Stack

- **Architecture**: MVVM (Model-View-ViewModel) with Repository pattern
- **Language**: Kotlin
- **Dependency Injection**: Dagger Hilt 2.52
- **Networking**: Retrofit 2.11 + OkHttp 4.12
- **Reactive**: Kotlin Coroutines 1.9 + RxJava 3
- **Database**: Room 2.6
- **Image Loading**: Glide 4.16
- **UI**: Material Design 3, ViewBinding
- **Min SDK**: 24 | **Target SDK**: 35

## Features

- Fetches cat facts from [Cat Facts API](https://alexwohlbruck.github.io/cat-facts/)
- MVVM with strict SOLID principles
- Offline caching with Room database
- Clean separation of concerns
- Hilt dependency injection

## Getting Started

```bash
git clone https://github.com/Zulqurnain/CatFactsMVVM.git
cd CatFactsMVVM
# Open in Android Studio and run
```

## Requirements

- Android Studio Hedgehog or newer
- Android SDK 35
- Kotlin 2.0.21
